# Welcome to your Expo app (THE INSPIRE QOUTE)

INSPIRE is a React Native application for finding quotes, personal reflection, and learning new words and it is built using Expo and provides a simple user experience for managing inspirational content.

## What API did you use?
I used the FavQs API, a public API that provides a collection of quotes. The app integrates it using a secure token to fetch, filter, and search through thousands of inspirational messages. FavQs allows you to collect and discover favorite quotes where you use apiKey. 

## What problem does your app solve?
The app called INSPIRE is made to help people feel motivated every day and improve their mental health. It gives users a place where they can find inspiring quotes, save their favorite ones, and write their own positive messages to help them stay happy and motivated everyday. It can also serve as a space where users can express their thoughts or even their rants. It allows them to write things they may not be able to share with others and it can also be a source of personal inspiration. Everything stays inside the app so it remains private to the user.

## What was the most difficult part of the integration?
The hardest part was making the online data and the user’s saved data work together properly. I needed to make sure that if a user liked a quote, it would stay liked even after searching again or refreshing the app. The Dictionary feature was also very difficult for me because I could not fetch the data properly when I searched for a word. The meaning did not appear, and I was s running out of time so I had a hard time fixing it. I was running out of time so I was not able to include the part where users can add quotes. In that part, the quotes added by the user would disappear when they log out. The favorites feature is working fine but I only had a problem with the add quotes feature where it would disappear.

## What would you improve with more time?
I would improve the app by adding proper data saving so that user-added quotes will not disappear after logging out. I would fix and improve the Dictionary feature so that it can fetch and display word meanings. I would enhance the part where user can still access their saved quotes and personal notes if the log in again without disappearing those qoutes. I would improve the design and performance of the app to make it faster and easier to use.
